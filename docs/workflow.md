# ワークフロー

## ワークスペースの範囲

- `workspace_notes/`、`workspace_experiments/`、`docs/` は、ワークスペース全体向けの置き場
- コンペごとの反復作業は `competitions/<competition_name>/` の中に置く

## 推奨するコンペ構成

- `competitions/<competition_name>/notes/`
  - 日本語のラフメモや調査ログ

- `competitions/<competition_name>/experiments/`
  - そのコンペの実験まとめ

- `competitions/<competition_name>/docs/`
  - そのコンペ固有の安定した参照資料、ルール、仮定

- `competitions/<competition_name>/experiment_log.md`
  - そのコンペの実験履歴の本体

## ルート側を使う場面

- `workspace_notes/` は、ワークスペース全体のアイデア、再利用したい思考、メタメモに使う
- `workspace_experiments/` は、コンペ横断の比較や共有ベンチマークに使う
- `docs/` は、リポジトリ全体の規約や運用ルールに使う
