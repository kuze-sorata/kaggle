# ドキュメント

このフォルダには、ワークスペース全体で使う安定した参照資料を置きます。

## 目的

- 詳細な運用ルールを `AGENTS.md` から分離する
- 共有しやすい規約や作業手順をまとめる
- 人間とエージェントの両方が参照しやすい安定資料を置く
- 雑多なメモ置き場にはしない

## 現在のファイル

- `CONVENTIONS.md`: `notes` / `experiments` / `docs` の使い分け
- `workflow.md`: ワークスペース全体とコンペ単位の運用
- `experiment_logging.md`: 実験ログの書き方
- `naming_conventions.md`: 命名ルール
